## 🐛 Troubleshooting

### Site Not Loading?
1. Check deployment status in Vercel dashboard
2. Verify URL is correct
3. Clear browser cache (Cmd+Shift+R)

### Changes Not Showing?
1. Deploy to preview first: `vercel`
2. Check preview URL works
3. Then deploy to production: `vercel --prod`

### Build Failed?
1. Check Vercel dashboard for error logs
2. Test locally first: `python3 -m http.server 8000`
3. Fix errors and redeploy

## 💡 Vercel Commands Reference

### Deployment Commands
```bash
# Deploy to production
vercel --prod

# Deploy to preview (staging)
vercel

# Deploy with specific project name
vercel --name ihw

# Deploy and skip questions
vercel --prod --yes

# Check deployment status
vercel ls

# View logs
vercel logs
```

### Project Management
```bash
# List all projects
vercel ls

# Remove project
vercel remove ihw

# Get project info
vercel inspect ihw
```

### Alias Management
```bash
# Set custom alias
vercel alias set <deployment-url> <custom-url>

# List aliases
vercel alias ls

# Remove alias
vercel alias rm <alias>
```
