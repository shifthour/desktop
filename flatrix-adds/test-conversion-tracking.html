Google Ads Conversion Tracking Test

How to get your conversion label:

  1. Go to Google Ads
  2. Click Tools & Settings → Conversions
  3. Click "+ New conversion action"
  4. Choose "Website"
  5. Set up as "Submit lead form"
  6. Copy the conversion ID (format: AW-17340305414/abcDEF123)

To verify tracking works:

  1. Install Google Tag Assistant Chrome extension
  2. Visit your website
  3. Submit a test form
  4. Check if conversion fires in Tag Assistant
