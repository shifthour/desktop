  CLAUDE API PROMPT TEMPLATE
  For Generating Business Listing Landing Pages from PDF Teasers

  HOW TO USE:
  1. Upload the business teaser PDF to Claude

PROMPT STARTS BELOW:
--------------------

You are a senior frontend developer and investment banking designer. I am uploading a business teaser PDF for a business listing. Analyze every detail in this PDF — company profile, financial data, investment highlights, service breakdown, industry dynamics, growth opportunities, reason for divestiture, contact information, listing ID, and project name.

Generate a COMPLETE, SINGLE-FILE index.html landing page that presents this business listing as a premium investment opportunity. The page must be a self-contained HTML file with all CSS and JavaScript inline (no external files except CDN libraries and one logo image).

=== TECHNICAL STACK (USE EXACTLY THESE) ===

- Font: Google Fonts — Montserrat (weights: 300,400,500,600,700,800,900)
- CSS Framework: Tailwind CSS via CDN (https://cdn.tailwindcss.com)
- Charts: Chart.js v4.4.1 via CDN
- Animations: GSAP 3.12.2 + ScrollTrigger via CDN
- No build step, no npm, no frameworks — pure HTML/CSS/JS in one file
- The only external file is `logo-dark.png` (company logo placed in same directory)

=== COLOR PALETTE ===

- Primary Blue: #4AADE8
- Darker Blue: #2B7CB5
- Black: #000
- Background Light: #f8f9fb
- Card Background: #ffffff
- Card Border: #e2e7ed
- Text Dark: #000 (headings), #333 (body), #555 (secondary), #818181 (muted), #aaa (subtle)
- Success Green: #4CAF50

=== DESIGN SYSTEM ===

Typography:
- All text: font-family 'Montserrat', sans-serif
- Section labels: 12px, font-weight 700, letter-spacing 2px, color #4AADE8, uppercase
- Section headings: 2.2rem, font-weight 800, color #000
- Card titles: 14-16px, font-weight 700, color #000
- Body text: 13-15px, color #555 or #818181, line-height 1.6-1.8

Cards:
- Background: #ffffff
- Border: 1px solid #e2e7ed
- Border-radius: 16px
- Box-shadow: 0 2px 12px rgba(34,56,101,0.05)
- Hover: shadow increases, border darkens, translateY(-2px)
- Clickable cards open detail modals on click

Icon Boxes:
- Size: 48x48px (regular), 56x56px (growth section), 36x36px (industry section)
- Background: #000 (black)
- Border-radius: 12-14px
- SVG icons inside: stroke color #4AADE8, stroke-width 2, fill none
- All SVGs must have viewBox="0 0 24 24" regardless of rendered size

Buttons:
- Primary (btn-gold): background #000, color #fff, padding 12px 28px, border-radius 8px
- Outline (btn-outline): border 2px solid #000, color #000, transparent bg; hover: bg #000, color #fff
- Advisor (btn-advisor): background #4AADE8, color #fff; hover: background #2B7CB5
- All buttons: hover translateY(-2px) with box-shadow transition

=== PAGE SECTIONS (GENERATE ALL OF THESE IN THIS ORDER) ===

   - Background: #000, height ~115px
   - Logo image right-aligned (logo-dark.png, height 110px)

2. HERO SECTION (Dark Gradient)

   - Listing ID text above the project badge (small, grey, letter-spacing)

   - Large animated heading (3.5rem, weight 800) — business name split across 3 lines, last line in #4AADE8
   - Subtitle paragraph describing the business (18px, color #999)
   - 4-column stat grid with key metrics (revenue, margin %, growth %, operational stat)
     - Each stat: semi-transparent dark card, metric in 1.8rem #4AADE8, label in 11px #888
     - Animate numbers counting up on scroll (IntersectionObserver)
   - "Scroll to explore" indicator at bottom
   - IMPORTANT: Add themed particle animation in background (e.g., snowflakes for winter business, leaves for landscaping, money signs for finance, etc.) — make it relevant to the business type

3. COMPANY PROFILE (White Background)

     LEFT: Doughnut chart (Chart.js)

       - Each card: flex:1 to stretch to match chart height

=== ANIMATIONS & INTERACTIONS ===

- GSAP ScrollTrigger: Fade-in + slide-up for each section (opacity 0→1, y 30→0)
- Hero: Timeline animation — badge slides in, title slides from left, subtitle fades up, CTA fades up
- Number counters: IntersectionObserver triggers count-up animation on stat elements
- Progress bars: Animate width from 0 to data-width% on scroll into view
- Particle background: CSS animation with random positioning, varying sizes, speeds, and opacity
- Cards: Hover lift (translateY -2px) + shadow increase
- Accordion: Max-height 0 → 500px transition for expand/collapse
- Smooth scroll for any anchor links

- Section padding: 40px 20px
- All grids: stack to single column or 2 columns
- Hero stats: 2x2 grid
- Hero title: 2rem
- CTA layout: stack vertically
- Floating bar: smaller padding, wrap buttons

@media (max-width: 480px) — Mobile:
- Navbar: height 60px, logo 48px
- Section padding: 30px 16px
- All grids: single column
- Hero stats: single column
- Hero title: 1.5rem
- Floating bar: short button labels
- Contact popup: full-width with margin

=== IMPORTANT RULES ===

1. Extract ALL data from the PDF — every number, percentage, text block, highlight
2. Calculate additional metrics if possible (CAGR, growth rates, margins)
3. All SVG icons must have viewBox="0 0 24 24" even if rendered smaller
4. Use semantic, descriptive modal IDs (e.g., 'inv-revenue', 'fin-cagr', 'svc-snow')
5. Modal content should EXPAND on what the PDF says — add context, breakdowns, implications
6. Financial chart data must exactly match the PDF numbers
7. Differentiate projected/estimated years (e.g., 2026E, 2027E) with lighter colors in charts
8. All external links open in new tab (target="_blank")
9. Add appropriate meta title and description for SEO
10. Custom scrollbar styling (thin, subtle)
11. Include body overflow-x: hidden to prevent horizontal scroll
12. All text must come from the PDF — do not invent business details
13. The page alternates between white (#fff) and light gray (#f8f9fb) section backgrounds
14. Logo image uses src="logo-dark.png" with onerror fallback

=== OUTPUT ===

Generate the COMPLETE index.html file. Do not truncate or summarize. Include every section, every modal, every animation, every responsive breakpoint. The file should be production-ready and deployable as-is with just the logo image file alongside it.
