## ✏️ How to Customize

### **1. Change Company Name**

Open `index-new.html` and find (around line 250):
```html
ISHTIKA <span class="text-gold-500">HOMES</span>
```

### **2. Update Phone Number**

Find all instances of:
```html
+91 98765 43210
```
And replace with your actual number.

### **3. Update Email**

Find:
```html
info@ishtikahomes.com
```
And replace with your email.

### **4. Change Project Details**

Scroll to the "Projects Section" (around line 800) and update:
- Project names
- Locations
- Prices
- Configurations
- Images

### **5. Replace Images**

Current images are from Unsplash (free stock photos).

Find:
```html
<img src="https://images.unsplash.com/..." alt="...">
```

Replace with:
```html
<img src="images/your-photo.jpg" alt="...">
```

(Put your photos in a folder called `images` next to the HTML file)
